# DJMol

### System Requirements

+ **Software**: 
    + Windows 10-7 64 bit OS
    + Java 1.8 or later, Python 3.x

+ **Hardware**:
    + At least 20 GB free space to install DJMol and add-ons. 
    + Preferred RAM size is 4 GB or higher.
